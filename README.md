# Urban Fashion 👗🛍️

## 📌 Description

Urban Fashion is a full-stack web application that allows users to browse fashion products and interact with a modern user interface. The application includes features like product listing and basic cart functionality. It is built using React for the frontend and Node.js with Express for the backend, providing a smooth and responsive user experience.

---

## 🚀 Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB

---

## ✨ Features

* Product Listing
* Add to Cart
* Responsive UI
* Basic API Integration

---

## 📂 Project Structure

* frontend/ → React frontend
* backend/ → Node/Express backend

---

## ⚙️ How to Run Locally

### 🔹 Backend

cd backend
npm install
node server.js

### 🔹 Frontend

cd frontend
npm install
npm start
